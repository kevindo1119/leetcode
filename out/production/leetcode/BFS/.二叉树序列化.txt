什么是序列化(Serilization)？
  将内存中结构化的数据变成字符串的过程
  序列化：object to string
  反序列化: string to object
什么时候需要序列化？
  1. 将内存中的数据持久化存储的时候
      内存中重要的数据不能只是呆在内存里， 断电九没有了，需要用一种方式写入硬盘，在需要的时候能否再从硬盘中读出来在内存中重新创建
  2. 网络传输时
      机器鱼机器之间交换数据的时候，不可能互相去读对方的内存。只能将数据变成字符串数据通过网络传输过去。接受的一方再将字符串解析后放到内存中
常用的一些序列化手段
  xml
  json
  thrift(facebook)
  protobuf(google)
  
序列化算法
  -压缩率 thrift protobuf
  -可读性 jason
